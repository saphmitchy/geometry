# geometry

競技プログラミングのための幾何ライブラリ

## ファイル構成
- [src](src) -- ライブラリソースファイル
- [test](test) -- テストコード
- [AOJ](AOJ) -- Aizu Online Judge での verify 用提出コード
  - [expander.py](AOJ/expander.py) -- ライブラリを展開するスクリプト

## テストについて
[Google test](https://github.com/google/googletest) を利用しています。
```
./run_test
```
などで実行できます。
